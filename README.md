# Airflo Systems

A frontend web application built using ReactJS.

This project was developed as part of a client delivery and focuses on providing a clean and simple user interface.

---

## Tech Stack

- Frontend: ReactJS  
- Hosting: Firebase Hosting  

---

## Live URL

http://localhost:3000/

Note: This URL is for local development. The project can be deployed to Firebase Hosting for production use.

---

## Setup & Installation

1. Clone the repository
2. Install dependencies  
   npm install
3. Run the project locally  
   npm start

The application will run on http://localhost:3000/

---

## Firebase Hosting

This project is configured to be hosted using Firebase Hosting.

To deploy updates:
```bash
npm run build
firebase deploy
